## Changes
**Files Created**:
- `backend/utils/code_metrics.py` — Utility module with AST-based code analysis functions:
  - `count_lines()`, `count_functions()`, `count_classes()` — basic metrics
  - `parse_imports()` — dependency extraction
  - `detect_long_functions()`, `detect_deep_nesting()`, `detect_too_many_parameters()` — smell detection
  - `calculate_complexity_score()` — composite health score
  - `analyze_project()` — full project scan with configurable thresholds

- `backend/tools/code_health_analyzer.py` — Tool implementation:
  - Inherits `BaseTool`
  - `name='code_health_analyzer'`
  - `parameters_schema`: project_path (required), max_function_lines, max_nesting_depth, max_parameters (optional thresholds)
  - `execute()` returns dict with `status`, `results` (metrics, smells, score), and `markdown_report`

- `tests/test_code_metrics.py` — 10 unit tests for utility functions
- `tests/test_code_health_analyzer.py` — 7 integration tests for tool

**Files Modified**:
